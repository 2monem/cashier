# 🛒 كاشير Pro — نظام نقطة البيع

نظام كاشير بسيط وسريع لمحلات منتجات العناية بالمنزل والجسم.

## ✨ المميزات

- 🛒 **شاشة البيع** — أضف منتجات للفاتورة بكليك واحد
- 💰 **حساب تلقائي** — إجمالي، خصم، باقي من المدفوع
- 📦 **إدارة المنتجات** — إضافة / تعديل / حذف
- 📊 **تقارير** — مبيعات اليوم وكل الوقت
- 💾 **حفظ تلقائي** — البيانات محفوظة في المتصفح

## 🚀 طريقة التشغيل

### محلياً (بدون إنترنت)
فقط افتح ملف `index.html` في المتصفح — وخلاص!

### على GitHub Pages (مجاني)
1. اعمل repository جديد على GitHub
2. ارفع ملف `index.html`
3. روح **Settings → Pages → Branch: main**
4. هيطلع لك رابط مجاني مثل: `https://username.github.io/cashier`

## 📱 يشتغل على
- ✅ Chrome / Firefox / Edge
- ✅ موبايل وتابلت
- ✅ بدون إنترنت بعد أول تحميل
